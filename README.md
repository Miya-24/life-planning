# life-planning

このリポジトリは、自分の人生の目標と日々の行動を GitHub で管理するための場所です。

## 🎯 目的

このリポジトリは、次の中長期目標を着実に達成するために使います。

- 応用情報技術者試験（2025年秋）に合格する
- Azure Fundamentals を2025年中に取得する
- 英語学習を習慣化する
- ジムとテニスで健康を維持・向上させる
- ベース演奏のスキルを高める

これは単なる TODO リストではありません。  
**長期目標を、毎週実行できる行動に変換するための仕組み**です。

---

## 🧠 このリポジトリの運用方法

このリポジトリは、次の3つのレイヤーで管理します。

### 1. Milestone = 長期目標
各資格や分野（応用情報、Azure、英語、運動、ベース）を Milestone として管理します。

### 2. Project = 週の計画
1週間につき1つの GitHub Project を作成します（例：`2026-W02`）。
その週にやることはすべてこの Project に入れます。

### 3. Issue = 実行可能な行動
Issue は 1回の作業（30〜90分程度）で完了できる粒度で作ります。  
例：  
- 「応用情報 午前 問1〜20」  
- 「Azure モジュール1」  
- 「ジム（上半身）」  

---

## 📅 週次運用

毎週、次のサイクルで運用します。

1. 新しい週の Project を作る  
2. その週にやる Issue を登録する  
3. 1週間かけて Issue を消化する  
4. 週末に振り返りを行い、次週の計画を調整する  

進捗は「やった時間」ではなく、**完了した Issue の数と中身**で判断します。

---

## 🛠 運用ルール

- Issue は小さく、必ず完了できるサイズにする
- 気合ではなく仕組みで回す
- うまくいかなかったら、自分ではなく「運用」を改善する
