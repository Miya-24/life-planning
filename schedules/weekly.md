# 週間スケジュール

AP（応用情報）を最優先とし、テニスのパフォーマンスを高めるためのトレーニングを組み込んだ構成。

| 曜日 | heavy (30-90min) | light (15-60min) | 備考 |
| :--- | :--- | :--- | :--- |
| **月** | **AP学習** | 英語 | |
| **火** | **ジム Day 1** | Azure | ジム(店舗)優先の日 |
| **水** | **AP学習** | **自宅ケア(Day 4)** | 学習優先。移動時間を削る |
| **木** | **AP学習** | 英語 | |
| **金** | **Azure集中** | ベース | |
| **土** | **ジム Day 3** | AP復習 / 英語 | 最大強度のトレーニング |
| **日** | **テニス** | 週間レビュー | 14:00~ テニス本番 |

## 運用ルール
- **heavyは1日1つまで**（脳のheavyと体のheavyが重なる日は、どちらかを軽めにする）
- **「今日の一歩」に集中する**（秘書：結城さんのルールを適用）
- できない日があっても良い。Issueに「今日は無理でした」と書いてCloseする。
