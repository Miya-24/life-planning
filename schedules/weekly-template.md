## 🛠 heavy/light の定義

- **heavy**：30〜90分で完了する集中タスク  
  - AP → 午前/午後フェーズ1タスクや復習  
  - Azure → Learn モジュールや演習  
- **light**：20〜40分程度の習慣タスク  
  - 英語 → シャドーイングや文法  
  - ベース → 曲の練習  
  - 休養 → 軽い復習・ストレッチ
