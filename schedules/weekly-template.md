# 週間テンプレ

| 曜日 | heavy | light |
|------|------|------|
| 月 | AP | 英語 |
| 火 | Azure | ベース |
| 水 | AP | 英語 |
| 木 | Azure | 休養 |
| 金 | AP | ベース |
| 土 | AP + Azure | ジム |
| 日 | なし | テニス・レビュー |

## ルール
- heavy は1日1個
- 日曜は heavy をやらない
- 平日は基本「ジムなしテンプレ」を使う
- 週1回だけ「ジムありテンプレ」を使う
- ジムの日は light（英語・ベース）はやらない
- heavy は 45分でOK
