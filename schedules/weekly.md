# 週間スケジュール

AP（応用情報）を最優先とし、テニスのパフォーマンスを高めるためのトレーニング（Day 1-4）を組み込んだ構成です。

| 曜日 | heavy (30-90min) | light (15-60min) | 備考 |
| :--- | :--- | :--- | :--- |
| **月** | **AP学習** | 英語 | 週の始まりに最大リソースを投入 |
| **火** | **ジム Day 1** | Azure | 体のパワー系。Azureは動画視聴など軽めに |
| **水** | **AP学習** | ジム Day 4 | 脳を使い、体は「整え」でリフレッシュ |
| **木** | **AP学習** | 英語 | 学習の連続性を維持 |
| **金** | **Azure** | ベース | Azureを一気に進める枠 |
| **土** | **ジム Day 3** | AP復習 / 英語 | 脂肪燃焼でテニス用のスタミナ強化 |
| **日** | **テニス** | 週間レビュー | 本番。脳は休めて振り返りを行う |

## 運用ルール
- **heavyは1日1つまで**（脳のheavyと体のheavyが重なる日は、どちらかを軽めにする）
- **「今日の一歩」に集中する**（秘書：結城さんのルールを適用）
- できない日があっても良い。Issueに「今日は無理でした」と書いてCloseする。
