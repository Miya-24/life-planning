# Azure Fundamentals 学習戦略

## 目的
- 今年中に Azure Fundamentals を取得する
- クラウドの基礎理解と試験範囲の習熟

---

## 基本方針
- Microsoft Learn のラーニングパスを中心に学習
- Udemy の動画で補足
- 演習問題を通して理解を定着させる

---

## 学習リソース

### Microsoft Learn
- [Azure Fundamentals ラーニングパス](https://learn.microsoft.com/ja-jp/training/courses/az-900t00)

### Udemy
- [Microsoft Azure AZ-900 Exam Prep](https://www.udemy.com/course/microsoft-azure-az900-exam)
- [Microsoft Azure AZ-900 Video Course](https://www.udemy.com/course/microsoft-azure-az900-video)

---

## 学習フェーズ

### フェーズ1：基礎理解
- Microsoft Learn のモジュール順に学ぶ
- 基本用語・概念を理解

### フェーズ2：演習
- Learn の演習問題を全て実施
- Udemy 動画で不足分を補う

### フェーズ3：復習と模擬試験
- モジュールの理解度チェック
- 弱点分野を重点復習

---

## GitHub運用との対応
- モジュール学習 → `azure + heavy`
- 演習問題・復習 → `azure + light`
