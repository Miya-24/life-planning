# 応用情報（AP）フェーズ別 Issue 一覧テンプレ

このドキュメントは、  
**AP 学習の Issue を作る際の「選択肢一覧」**である。

- すべてを一気に Issue 化しない  
- その週に必要な分だけ選んで Issue にする  
- フェーズと分野を意識して進める  

---

## フェーズ0：立ち上げ・全体把握（初週のみ）

※ AP 学習開始時、または長期間空いた後に使用

- AP 午前：ネットワーク基礎理解（フェーズ1）
- AP 午前：DB基礎理解（フェーズ1）
- AP 午前：セキュリティ基礎理解（フェーズ1）
- AP 午後：ネットワーク問題構造把握（フェーズ1）
