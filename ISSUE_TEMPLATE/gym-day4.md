## 🧘 入念なケア（共通）
- [ ] ローラーリリース（15分：癒着を剥がす）
- [ ] 膝つけ壁ストレッチ（記録：右__cm / 左__cm）

## 🏋️ 調整メニュー（いずれかを選択）

### 【店舗ジムの場合】
- [ ] インナーサイ（20回×2：最軽量）
- [ ] アウターサイ（20回×2：丁寧）
- [ ] バックエクステンション（15回×2）
- [ ] ラテラルレイズ（15回×2：軽め）

### 【自宅の場合】
- [ ] **床バックエクステンション**（15回×2）
- [ ] **アームスイング（前後・左右）**（各20回）
- [ ] **アームスイング（上下）**（20回）
- [ ] **肩甲骨回し（前後）**（各20回）
- [ ] **キャット＆カウ**（15回）
- [ ] **股関節回し（四つん這い）**（各15回）
- [ ] **ダイナミック・ランジ**（各10回）

## 🏃 有酸素
- [ ] クロストレーナー20分 または 散歩15分
