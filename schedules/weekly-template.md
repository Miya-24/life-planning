## 🛠 heavy/light の定義

- **heavy**：30〜90分で完了する集中タスク  
  - AP → 午前/午後フェーズ1タスク  
  - Azure → Learn モジュールや演習  
  - 週間レビュー（軽め heavy） → 週の振り返り・次週計画作成  
- **light**：15〜60分程度で実施する習慣タスク  
  - 英語 → シャドーイングや文法  
  - ベース → 曲の練習  
  - ジム → 軽い運動やストレッチ  
  - AP復習 → 前日・週末学習内容の整理・確認  
  - テニス → 運動習慣

---

## 🔁 週間運用ルール

1. 各日 heavy は **1枠のみ（週末はまとめ可）**  
2. light は heavy の負荷や運動の有無に応じて柔軟に割り振る  
3. 土曜は heavy をまとめて消化可（AP 午後＋Azure 演習）、light に AP復習 + ジム  
4. 日曜は **軽め heavy：週間レビュー** + light にテニス + AP復習  
5. heavy/light の枠は **曜日ごとに固定**、Issue の内容だけ更新する
